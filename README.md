# Игра - Поиск предметов

## Проект построен по модульному принципу. Каждый модуль отвечает за свою часть логики.

- config.js - Хранит массив предметов, а также параметры уровней сложности. Легко добавлять новые предметы и уровни.

- flashlight.js - Реализует затемнение через слой overlay. Движение фонарика за курсором.

- gameState.js - Содержит состояние (какие предметы найдены) и методы для его изменения (itemFound, getItemFound, allCheckFound).

- go.js – отвечает за стартовый экран с выбором уровня и отображением сохранённых звёзд. При клике на уровень запускает игру с соответствующими параметрами.

- itemGenerate.js - Создаёт DOM элементы для каждого предмета и размещает их со случайными координатами.

- itemManager.js - Вешает обработчик кликов на комнату, определяет, по какому предмету кликнули, обновляет состояние и интерфейс, проигрывает звуки, а также считает промахи.

- sound.js - Создаёт аудиоэлементы для всех звуков, предоставляет методы для взаимодейстия с аудио.

- star.js - работа со звёздами: получение сохранённых данных из localStorage, сохранение звёзд за уровень, расчёт количества звёзд на основе промахов.

- main.js - инициализирует все модули в правильном порядке, связывает их, запускает игру с параметрами выбранного уровня, обрабатывает победу и поражение, сохраняет результат.

- timer.js - управляет таймером уровня: запуск, остановка, обновление отображения, скрытие. При истечении времени вызывает колбэк проигрыша.

- ui.js - управляет отображением списка предметов на боковой панели, счётчиком, обновляет интерфейс при нахождении предметов.

## Особенности реализации

- Модульность - каждый файл решает одну задачу, код легко читать и расширять.

- Изоляция состояния - gameState не знает о DOM, что упрощает тестирование.

- SOLID - принципы соблюдены: единственная ответственность, открытость/закрытость (можно добавлять предметы без изменения логики), инверсия зависимостей.

## Линейна интерполяция

Для следования фонарика за курсорои использовалась линейная интерполяция, т.е. каждый кадр текущие координаты фонарика приближались к целевым на 30%, тем самым дало плавное передвижение фонарика и низкую нагрузку

## Геймплей

- Уровни сложности - три уровня (лёгкий, средний, сложный) с разным фоном, размером предметов и лимитом времени. Будет не легко найти все предметы на последнем уровне)

- Таймер - для уровней с ограничением по времени ведётся обратный отсчёт, при окончании времени игра завершается.

- Система звёзд - результат прохождения оценивается звёздами (3, 2 или 1) в зависимости от количества промахов. Звёзды сохраняются в localStorage и отображаются в меню.
